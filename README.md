-----Habit Tracker API-----
A professional Habit Tracker REST API built using FastAPI. Users can register, login, create habits, track daily completions, monitor streaks, and view productivity analytics.


----###_Features_###----
User Registration
JWT Authentication
Password Hashing
Create Habit
View Personal Habits
Complete Habit
Current Streak
Longest Streak
Calendar View
Habit Dashboard
User Dashboard
Productivity Score
Achievement Badges
User Levels
Personalized Insights
Heatmap Data
Interactive Swagger Documentation


----###_Tech Stack_###---
Python
FastAPI
SQLAlchemy ORM
MySQL
JWT Authentication
Passlib (Password Hashing)
Pydantic
Uvicorn


----###_Project Structure_###----

app/
 ├── core/
 ├── database/
 ├── models/
 ├── routers/
 ├── schemas/
 ├── services/
 └── main.py


----###_Authentication_###----
Protected routes use JWT Bearer Authentication.
